# Changelog — ARISTOTEL Textbook

История релизов учебника. В этом файле фиксируются только публичные изменения:
выпуски новых глав, существенные правки в опубликованном материале, изменения
в структуре сайта. Внутренняя работа по разметке базы знаний фиксируется
в приватном репозитории `aristotel-kb`.
